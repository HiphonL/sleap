class sleap.io.video.Video(backend: Union[sleap.io.video.HDF5Video, sleap.io.video.NumpyVideo, sleap.io.video.MediaVideo])[source]¶

The top-level interface to any Video data used by sLEAP is represented by the Video class. This class provides a common interface for various supported video data backends. It provides the bare minimum of properties and methods that any video data needs to support in order to function with other sLEAP components. This interface currently only supports reading of video data, there is no write support. Unless one is creating a new video backend, this class should be instantiated from its various class methods for different formats. For example:

>>> video = Video.from_hdf5(file='test.h5', dataset='box')
>>> video = Video.from_media(file='test.mp4')
